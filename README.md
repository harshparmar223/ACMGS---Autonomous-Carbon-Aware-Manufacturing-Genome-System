# ACMGS вЂ” Autonomous Carbon-Aware Manufacturing Genome System

## Overview
ACMGS is an intelligent manufacturing optimization system that combines:
- **Energy DNA Modeling** вЂ” LSTM Autoencoder learns machine energy signatures
- **Batch Genome Encoding** вЂ” Unified feature vectors for every production batch
- **Evolutionary Optimization** вЂ” NSGA-II multi-objective optimization (yield, quality, energy, carbon)
- **Carbon-Aware Scheduling** вЂ” Adjusts production decisions based on grid carbon intensity

## How Components Interact
1. **Simulator** generates realistic manufacturing data в†’ saved to `data/simulated/`
2. **Energy DNA** trains on machine power signals в†’ produces embeddings + saved model
3. **Batch Genome** merges process params + energy embeddings + material + carbon в†’ genome vectors
4. **Predictor** trains on genome vectors в†’ predicts yield, quality, energy
5. **Optimizer** uses predictor as fitness evaluator в†’ finds Pareto-optimal configurations
6. **Scheduler** picks best configuration based on real-time carbon intensity
7. **Database** stores all records for traceability
8. **API** exposes endpoints for external integration
9. **Dashboard** visualizes everything in real-time
